# AiStudio

Assignment data and projects from AI Studio coursework.

## Overview

This repository contains hands-on projects and assignments completed as part of the AI Studio module, covering:

- Neural network architectures
- Model training and evaluation
- Hyperparameter tuning
- Real-world AI application design

## Tech Stack

- Python
- TensorFlow / Keras
- Jupyter Notebooks
- NumPy, Pandas, Matplotlib

## Getting Started

```bash
git clone https://github.com/panchalaman/AiStudio.git
cd AiStudio
pip install -r requirements.txt
```
